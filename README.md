# Hazel

[nugu.jp](https://www.nugu.jp) 쇼핑몰의 상품 성과 지표를 실시간으로 확인할 수 있는 크롬 익스텐션입니다.

## 주요 기능

- **상품 썸네일 위에 성과 지표 오버레이 표시**
  - 조회수 — `customerevent` API의 `view_item` 액션 이벤트 count
  - 구매수 — `orderitem` API의 주문 아이템 count
  - CVR(전환율) — 구매수 / 조회수 × 100 (클라이언트 계산)
- **기간별 조회** - 최근 30분 ~ 최근 30일까지 다양한 기간 선택 가능
- **스크롤 연동** - 화면에 보이는 상품만 자동으로 데이터를 로드 (디바운스 적용)

## 동작 방식

1. `www.nugu.jp` 페이지의 상단 네비게이션에 기간 선택 셀렉트박스를 추가합니다.
2. 기간을 선택하면, 현재 화면에 보이는 상품 코드를 수집합니다.
3. `admin.nugu.jp`의 관리자 API를 통해 조회/주문 데이터를 가져옵니다.
4. 각 상품 썸네일 위에 조회수, 구매수, CVR을 표시합니다.

> 관리자 API 호출을 위해 `admin.nugu.jp`에 로그인된 탭이 필요합니다. 로그인된 탭이 없으면 자동으로 어드민 페이지를 열어줍니다.

## 설치 방법

1. [ZIP 파일](https://github.com/mediquitous-dev/hazel/files/13649414/hazel-1.0.zip)을 다운로드하여 압축을 풉니다.
2. 크롬 브라우저에서 `chrome://extensions/`로 이동합니다.
3. 우측 상단의 **개발자 모드**를 활성화합니다.
4. **압축 해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다.
5. 압축 해제된 익스텐션 폴더를 선택하여 로드합니다.

## 사용 방법

1. `admin.nugu.jp`에 로그인합니다.
2. `www.nugu.jp`에서 상단 네비게이션의 **상품스탯** 셀렉트박스로 조회 기간을 선택합니다.
3. 상품 썸네일에 표시되는 조회수/구매수/CVR을 확인합니다.

## 프로젝트 구조

| 파일 | 설명 |
|------|------|
| `manifest.json` | 익스텐션 설정 (Manifest V3) |
| `coco.js` | `www.nugu.jp`에 주입되는 콘텐츠 스크립트 (UI 및 데이터 표시) |
| `noah.js` | `admin.nugu.jp`에 주입되는 콘텐츠 스크립트 (API 호출) |
| `background.js` | 두 탭 간 메시지 중계를 담당하는 서비스 워커 |
| `dayjs.js` | 날짜 계산용 라이브러리 |
| `popup.html` | 익스텐션 팝업 UI |
